## サイト概要
### サイトテーマ
施工管理1〜2年目向けの、工区（場所）ごとの工程・タスクをスマホで確認・共有できるタスク管理アプリです。
​
### テーマを選んだ理由
施工管理の現場では、口頭指示や突発対応が多く、特に若手ほど「全体の中で今どこまで進んでいるか」「次に何をすべきか」を把握しづらいと感じました。
その結果、タスクの抜け漏れや段取り不足が起きやすく、残業や精神的な負担につながり、早期離職の一因にもなっていると考えています。

そこで本アプリでは、工区（場所）単位で工程全体を見える化し、チェックリスト形式でタスクを整理・共有できる仕組みを用意することで、日々の行動を具体的にイメージできる状態を目指します。

特に電気工事などの設備業では、建築工程に合わせて電気の作業が決まり、作業が決まった時点で付随タスクが連鎖的に発生します。
例えば、材料の拾い出し・発注、搬入場所（置き場）の確保、搬入用台車の確保、揚重手配、搬入車両の時間調整、他業者との取り合い打ち合わせ、日々の作業予定への反映、図面作製・図面の配布確認などです。
しかし若手のうちは、この工程とタスクの連動性が見えにくく、結果として後手に回ることが少なくありません。

そこで、工程に応じたタスクのテンプレート化に加え、上司によるタスク精査・フィードバック機能を実装することで、属人的になりがちな段取りを仕組み化し、若手の教育効率を高めつつ、現場全体でカバーし合える体制の構築につなげたいと考え、本テーマで制作することにしました。


### ターゲットユーザ
施工管理として働き始めた1〜2年目で、業務の全体像を掴みにくい人
口頭指示や情報量の多さで、タスクの抜け漏れや優先順位付けに悩む人
週例（定例）で決めた内容を、日々の行動に落とし込むのが難しい人
​
### 主な利用シーン
朝一に、今日やるべき作業と期限が近いタスクを短時間で確認したい時
帰り際に、抜け漏れの確認と翌日の段取り（明日やること）を整理したい時
終礼で、工区ごとの工程とタスクを見ながら上司、担当・期限を決めて共有したい時
休み前や長期休暇前後に、再開時に迷わないよう現場状況と次の作業を整理したい時
